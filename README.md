# Oxygen Diffusion for Wound Healing

## Overview
This project simulates oxygen diffusion in wound healing using MATLAB. It helps analyze tissue oxygenation and its impact on the healing process.

## Files
- oxygen_diffusion.m – Main MATLAB simulation script
- wound_healing_details.m – Supporting functions and explanations

## Tools & Technologies
- MATLAB
- Biomedical Engineering concepts

## Applications
- Academic learning
- Biomedical research simulations
- Understanding wound healing dynamics
